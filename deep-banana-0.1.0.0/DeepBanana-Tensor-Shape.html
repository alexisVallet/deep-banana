type Size s :: Nat Source

The size, i.e. the total number of elements of a tensor of shape s, at the type level.

type Nbdim s :: Nat Source

The number of dimensions of a tensor of shape s. For instance:

  Nbdim '[] == 0
  Nbdim '[5] == 1
  Nbdim '[6,8] == 2
