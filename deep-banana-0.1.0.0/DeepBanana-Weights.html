newtype Weights a l Source

Weights for a neural network. Represents the weights as the concatenated HList of the individual layer's weights. The a parameter represents the weights scalar datatype, used for FixShape and VectorSpace instances.

class Floating (FixScalar t) => FixShape t where Source

Class for datatypes which are not Floating nor VectorSpace instances themselves, but can be lifted to fixed-shape versions which are through an error monad. Somewhat ad-hoc class Used to implement stochastic gradient descent in Optimize for non fixed shape datatypes in a type safe manner.
